# RPA 文件命名规范

> 维护人：骆铭源 | 生效日期：2026-04-08
> 本文档约束 RPA 导出文件的命名格式，入库脚本按此规范做文件匹配。
> **文件名不符合规范 → 入库脚本跳过 → 当天数据缺失！**

## 命名规则（必须遵守）

1. **分隔符统一用半角减号 `-`**，不要用括号（避免全角半角混乱）
2. **花括号 `{门店名}` / `{账号}` 替换为实际值**（如：小猴、善培臣），其余部分一字不改
3. **不要在文件名中加日期、长串 ID、公司全称**
4. **华信和槐荫可合并为"善培臣"导出**（用商家ID区分），其他门店不要随意合并
5. **不要改变文件格式**——CSV 就是 CSV，XLSX 就是 XLSX，不要互换
6. **目录下不要放无关文件**（如操作登记表、warn.docx 等）
