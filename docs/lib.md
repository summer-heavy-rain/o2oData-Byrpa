## 平台与数据源

**美团 / 饿了么 / 京东（财务对账通用铁律，🔄 2026-04-02 强化）**：

1. **先按订单号匹配**：订单明细 × 财务账单。**匹配到** → 当期结算；**订单明细有财务无** → **待结算**（展示数量，不入损益）；**财务有订单明细无** → **滞纳结算**（展示订单数+实结金额，入损益，未扣成本补扣）。
2. **实结金额逐单校验（阻断级）**：公式计算的实结必须与财务账单原始字段**逐单比对**。**美团**=`商家应收款`，**饿了么**=`结算金额`，**京东到家**=`应结金额`，**京东小时达**=`结算金额`。**任何一笔有差异** → **阻断该平台出数** + 提醒 xiaohou（差异笔数/金额/订单号样例）+ 等确认后才出数据。**禁止**静默跳过/补零/自行归项。
3. **RPA 数据缺失检测（阻断级）**：mini橙 API 有某日某平台订单但 RPA 导出为空（0 条）→ **阻断该平台出数** + 提醒 xiaohou 哪个平台异常 + 等确认。三平台独立检测，互不影响。

详见 `**docs/京东到家商家对账单验数.md` §1.1**。

- **层1 订单明细**：`ods_rpa_meituan_product_detail`（来自 `美团-商品明细（XX门店）.csv`，按商品一行一条，同一订单有多行）

## 订单与财务账单匹配归类（🔄 2026-04-02 确认）

每日数据处理时，订单明细和财务账单按订单号匹配，产生三种归类：

| 归类 | 条件 | 展示 | 是否入损益 | 成本处理 |
|------|------|------|----------|---------|
| **当期结算** | 订单明细有 & 财务有 | 正常展示 | ✅ 入损益 | 正常扣除 |
| **待结算** | 订单明细有 & 财务未出账 | 展示**待结算订单数量** | ❌ 不入损益 | 不入当期 |
| **滞纳结算** | 财务有 & 当日订单明细无 | 展示**订单数 + 实结金额** | ✅ 入损益 | 未扣过的**补扣** |

**待结算 → 滞纳结算流转**：后续某日财务账单出现该笔订单时，待结算数量 -1，该笔订单转入当日滞纳结算。

**当日门店毛利公式**：
```
当日毛利 = 当期实结 - 当期成本 + 滞纳实结 - 滞纳补扣成本（未扣过的）
```

**延迟退款**：财务有退款行但订单号不在当日明细 → 归入**滞纳结算**（负值，冲减当日毛利）。

## 退款处理（三平台通用规则）

**退款类交易行必须纳入结算计算**，不能只取正向订单。


| 平台  | 正向订单                              | 退款处理                                                                                               |
| --- | --------------------------------- | -------------------------------------------------------------------------------------------------- |
| 美团  | `交易类型=外卖订单`                       | `订单退款` / `订单部分退款` / `部分退款冲抵` **纳入**，按订单号匹配当日明细，匹配不到的归延迟退款                                          |
| 饿了么 | 财务 `"订单应收"`                       | 已取消/全退：**商户应收 >0 → 对账 0**；**<0 → 保留负值**（与 `o2o_eleme_order_fin_settlement_compare.sql` 一致）；赔偿行单独处理 |
| 京东  | 小时达=长表 `SUM(结算金额)` / 到家=宽表 `应结金额` | 退款类行纳入，按订单号匹配当日明细，匹配不到的归延迟退款                                                                       |

## 京东还原公式

**前置**：先看 `**订单来源`**（`ods_rpa_jd_order`）——**小时达** 与 **到家** 两套形态不同，**禁止混用**。

**小时达（长表）**：对账锚 `**SUM(结算金额)`** 按 `到家业务单号`（`ods_rpa_jd_finance` 行全加，符号已在列中）。**包装费（独立分项）**：`费用类型` ∈ {`**到家包装费`**, `**包装费**`}（字面以导出为准）**单独**做 `**SUM(结算金额)**`，**不计入**「商家补贴」分项（**账单未出该类型则当期该项不算**，见 `京东到家商家对账单验数.md` §1.1）。

**到家（商家对账单宽表）**：对账锚 `**应结金额**`；列名与恒等式见 `**docs/京东到家商家对账单验数.md**`。口述「商家承担消费」在导出中为 `**商家承担小费**`，须按表头修正映射。

**经营侧分项（小时达，`费用类型` 白名单）**——在样本「活动 / 商家补贴 / 财务运费 / 平台扣点」基础上，**包装费单独成项**（**不**并入商家补贴）：


| 样本列名    | 小时达 `费用类型` 归入（枚举见文档）                              |
| ------- | ------------------------------------------------- |
| 活动补贴    | 平台券/京东平台券/京豆/积分等                                  |
| 商家补贴    | 商家券、账扣款、限时抢购商家等（**不含** `到家包装费` / 包装类 `**包装费**` 行） |
| **包装费** | `**到家包装费**`、与导出一致的 `**包装费**`（单独 `SUM(结算金额)`）      |
| 财务运费金额  | 基础配送、远距离、满减运费、秒送基础服务费、免运券等                        |
| 平台扣点佣金  | 代收到家佣金、CPS 推广服务费等                                 |


**到家宽表**（与小时达分列）：货款=`订单原价`，活动=`平台承担补贴(市场费)`，运费相关=货补/运补/自送/小费/取件费等列之和，**仅此处**宽表 `**包装费**` 列（表头有则纳入应结），扣点=`总佣金(…)(可开票)`+`基础服务费`+`**阶梯扣点佣金**`；`**开票金额`**不入应结公式；**应结**=上列代数和（详见 `**docs/京东到家商家对账单验数.md`**）。`ods_rpa_jd_finance` 暂无此宽表结构时，以导出 xlsx 或专用入仓表验数。

**补贴类**行（部分类型）：`结算金额` = `京东承担金额`，`商家承担金额`=0；其余以 `**结算金额` 为准**。

**辅助校验（小时达长表）**：活动补贴白名单行 `结算金额`≈`京东承担金额`；脚本 `**sql/o2o_jd_xiaoshida_subsidy_jd_carry_validate.sql`**。

**辅助校验（到家宽表）**：`**scripts/validate_jd_daojia_merchant_statement.py`** 对「商家对账单」xlsx 跑应结恒等式。

## 结算 P&L 分类框架（三平台统一，2026-03-29）

### 瀑布结构

```
挂网价（商品总价/GMV）
 - 退款金额（全退 + 部分退）
 - 营销费
   - 主动营销（商家自主花钱：活动支出/霸王餐/CPS）
   - 被动营销（平台佣金/扣点，不可选择）
 ± 配送费（净值）
 + 包装费
 = 应结金额（公式可推导）

 + 平台赔付（平台赔给商家）
 - 罚款扣款（平台罚商家）
 ± 其他（保险/公益等）
 = 实结金额
```

### 京东已确认公式（🔄 2026-04-02 矫正）

**到家（宽表）应结金额**：

```
应结金额 = 订单原价
         + 平台承担补贴(市场费)
         + 商家承担货款补贴
         + 商家承担运费补贴
         + 商家自送配送费
         + 总佣金(货款佣金+运费佣金+餐盒费佣金)(可开票)
         + 取件服务费(开票)(正向单展示远距离运费;售后单则展示达达售后运费)
         + 商家承担小费
         + 基础服务费
         + 阶梯扣点佣金
         + 餐盒费
```

- 开票金额永不入公式
- 必须包含正向订单 + 退款订单 + 部分退款订单
- **实结金额** = 应结金额 + 不可推导收支项（支出项为负数）

**小时达（长表）应结金额**：

```
应结金额 = 用户实付货款
         + 活动补贴（平台）
         + 商家补贴
         + 包装费
         + 配送费
         + 平台扣点佣金
```

- 必须包含正向订单 + 退款订单 + 部分退款订单
- 费用类型归入：
  - 挂网价 = `用户实付货款`
  - 活动补贴（平台）= `平台券`/`京东平台券`/`京豆`/`积分`/`京东满减`/`全品类券`
  - 商家补贴 = `商家券`/`账扣款`/`限时抢购商家`等（**不含**包装费类型）
  - 包装费 = `到家包装费`/`包装费`（独立分项不入商家补贴）
  - 配送费 = `基础配送费`/`远距离运费`/`满减运费`/`秒送基础服务费`/`免运券`
  - 平台扣点佣金 = `代收到家佣金`/`CPS推广服务费`

### 三平台铁律：新子项必须确认

任何平台的财务账单出现**新增费用类型/新增子列/预留位开始填充数据**，AI **禁止自动纳入计算**，必须先通知 xiaohou 确认归类后才能加入公式。三个平台都适用。
